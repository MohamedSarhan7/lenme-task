# lenme-task

Clone the project
```bash
  git clone git@github.com:MohamedSarhan7/lenme-task.git
``` 
Go to the project directory

```bash
  cd lenme-task
```

create .env file

```bash
  cp .env.example .env
```

### using docker 
run docker compose

```bash
  docker-compose up
```
